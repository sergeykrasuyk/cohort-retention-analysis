# cohort-retention-analysis
Cohort analysis of user retention using SQL &amp; Google Sheets

Objective: To evaluate user retention rates by applying cohort analysis to identify behavioral patterns between promo and organic user groups.

Tools & Technologies: SQL (date parsing, table joins, cohort calculations, monthly offsets), Google Sheets (pivot tables, conditional formatting, dynamic addressing), Data Visualization.

Key Contributions:

Cleaned and transformed raw event data with various date formats using SQL queries to ensure data integrity.

Calculated user cohorts based on first activity date and determined user lifetime (tenure) within the company.

Built interactive cohort tables in Google Sheets, utilizing pivot tables and conditional formatting for clear visualization of retention trends.

Conducted comparative analysis of two user segments: Promo Users (acquired via promotions/ads) vs. Organic Users.

Formulated data-driven conclusions regarding acquisition quality, retention effectiveness, and provided actionable insights for marketing and product strategies.
